# Landing Registro 7mo Congreso No Más Malezas — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Construir y desplegar una landing page de registro con QR único + email automático para 1,200 asistentes del 7mo Congreso Internacional No Más Malezas de MAINTER.

**Architecture:** Página HTML monolítica embebida en WordPress vía REST API. CSS/JS servidos por jsDelivr desde GitHub. Form submit a webhook n8n self-hosted que persiste en Supabase, genera QR, renderiza PDF y envía email SMTP.

**Tech Stack:** HTML5 vanilla, CSS3 (variables), JS vanilla, GitHub (repo público), jsDelivr (CDN), n8n self-hosted, Supabase self-hosted, SMTP SiteGround, qrcode npm, puppeteer (PDF).

## Task 6: Workflow n8n registro

**Files:**
- Create: `n8n/workflow-registro.json`

- [ ] **Step 6.1: Crear workflow en n8n UI**

Login a `https://danzarte-n8n.rgxmhp.easypanel.host` con API key.

Crear nuevo workflow `NMM Registro`. Estructura nodos:

1. **Webhook** (trigger)
   - Path: `nmm-registro`
   - Method: POST
   - Response Mode: "When last node finishes"

2. **Function: Validate**
   ```javascript
   const { nombre, tipo, contexto, whatsapp, email } = $input.first().json;

   if (!nombre || nombre.length < 3) throw new Error('Nombre inválido');
   if (!['estudiante','productor','empresa','institucion','prensa','otro'].includes(tipo))
     throw new Error('Tipo inválido');
   if (!contexto || contexto.length < 2) throw new Error('Contexto requerido');

   const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
   if (!emailRegex.test(email)) throw new Error('Email inválido');

   const waClean = whatsapp.replace(/\D/g, '');
   if (waClean.length < 8) throw new Error('WhatsApp inválido');

   return [{ json: { nombre, tipo, contexto, whatsapp: waClean, email: email.toLowerCase().trim() } }];
   ```

3. **Function: Map tipo→color**
   ```javascript
   const map = {
     estudiante:  { color: 'azul',     punto: 'E', sector: 'Pre-emergente · ZethaMaxx' },
     productor:   { color: 'verde',    punto: 'N', sector: 'Barbecho · Ultracheval' },
     empresa:     { color: 'amarillo', punto: 'O', sector: 'Resistentes · Apresa' },
     institucion: { color: 'rojo',     punto: 'S', sector: 'Post-emergente · Balón' },
     prensa:      { color: 'blanco',   punto: '-', sector: 'Acceso prensa lateral' },
     otro:        { color: 'gris',     punto: '-', sector: 'General' }
   };
   const d = $input.first().json;
   return [{ json: { ...d, ...map[d.tipo] } }];
   ```

4. **Supabase: Check duplicate** (SELECT por email)
5. **IF: exists?** → si existe, ir a "Get existing QR" → "Respond"
6. **Supabase: Insert** (si no existe)
7. **Function: Generate QR payload**
   ```javascript
   const d = $input.first().json;
   const payload = JSON.stringify({ id: d.id, n: d.nombre, t: d.tipo, c: d.color });
   const qrcode = require('qrcode');
   const dataUrl = await qrcode.toDataURL(payload, { errorCorrectionLevel: 'H', width: 600 });
   return [{ json: { ...d, qr_data_url: dataUrl, qr_payload: payload } }];
   ```
8. **Function: Render PDF entrada** (puppeteer template) — ver email/pdf templates en Task 7-8
9. **SMTP: Send email** con HTML + PDF attached
10. **Supabase: Update email_enviado_at = NOW()**
11. **Respond**: `{ ok: true, id, color, punto, sector, qr_data_url }`
